# Todo APP

## 리액트를 이용한 할 일 웹 어플리케이션

![메인](./img/todo.png)

#### 링크: [TODO APP](https://jjjjhjjjj.github.io/Todo-App/)

#### 기능

    - 할 일 추가 (기본값 Doing)
    - 리스트 할 일 상태 카운트
    - 상태별로 리스트 구분
    - 상태 변환, 삭제 기능 구현
    - useReducer, ContextAPI를 이용한 상태 관리
